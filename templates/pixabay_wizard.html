Set Up Pixabay Image Search

Follow these steps to enable in-app image search for quiz questions. Pixabay provides free images with no attribution required.

1 Get API Key
2 Enter & Test
3 Done

Get a Free Pixabay API Key

  1. Go to pixabay.com/api/docs and create a free account (or sign in).
  2. After signing in, your API key will be displayed on that page.
  3. Copy the key and come back here to paste it in the next step.
Free and no credit card required. The Pixabay API is free for all use cases. There is a rate limit of 100 requests per minute, which is more than enough for classroom use.
Cancel

Enter and Test Your API Key

Paste your Pixabay API key below and test the connection.

What happens during the test? QuizWeaver sends a small test query to Pixabay to verify your key works. No lesson data is sent.

Pixabay Connected

You can now search for images directly from quiz question cards. Click the "Search" button on any question to find and attach images.
